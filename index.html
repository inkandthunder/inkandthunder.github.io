MLB Win-Loss Visualizer

A web application for visualizing the year over year win-loss records of Major League Baseball teams.

Technology

Historical data was gathered and parsed from Retrograph datasets using a .NET Core application written in C#, which converted the set into JSON organized by team and year. The year-over-year graphs are generated using Highcharts, and Bootstrap4 is the CSS framework of choice for this project

WAR Calculator

A simple MLB Wins Above Replacement (WAR) calculator. Users enter player position and a few known stats, and a valuation of approximate worth in "wins" is determined.

Technology

WAR Calculator is written entirely in R. On the backend, RStudio hosts the code and Shiny provides the front-end library. Formulations for MLB were found on FanGraphs.

Tappsy

Powered by data from Untappd's API, Tappsy is a web application that visualizes personal craft beer check-ins to examine information about beer tastes and drinking habits.

Technology

The dc.js (D3 and Crossfilter.js) libraries were used to create the interactive charts and corresponding data tables. A customizable, interactive map (that can either format Mapbox tiles or OpenStreetMap data) was built using Leaflet.
